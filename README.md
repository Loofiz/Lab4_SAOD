# Сервис сокращения ссылок (URL Shortener)

Упрощённая реализация сервиса, который позволяет преобразовывать длинные URL в короткие коды и восстанавливать исходные ссылки по короткому коду.

## Возможности

- Генерация уникального короткого кода для любой длинной ссылки
- Получение исходной ссылки по короткому коду
- Проверка существования короткого кода
- Вывод всех сохранённых пар (короткий код → длинная ссылка)
- Запрет создания дубликатов коротких кодов

## Как это работает

1. Пользователь передаёт длинную ссылку (например, `https://example.com/articles/python-basics`).
2. Сервис генерирует короткий код (например, `a1b2`) с помощью **base62-кодирования** счётчика.
3. Пара `короткий_код → длинная_ссылка` сохраняется в **хеш-таблице**.
4. При запросе по короткому коду сервис находит и возвращает исходную ссылку.
